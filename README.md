# Patient Referral Tracker — Frontend
A clean, professional React frontend for managing patient referrals across healthcare facilities.

# Tech Stack

React 18 + TypeScript
Vite
Tailwind CSS v4
Axios
React Router DOM
Lucide React (icons)

# Role Based UI

ADMIN                  all access         all access
DEVELOPER              all access         all access
FACILITY_ADMIN,        Own facility       Create, update status
REFERRAL_OFFICER,      All referrals      Create, update status
VIEWER                 All referrals      Read only


# Connected Backend
This frontend connects to the Patient Referral Tracker API.
